## Project Info

Manged in [Github Projects](https://github.com/users/Rabbit-89/projects/1)

This is a website for viewing, rating and commenting on recipes you can cook in your Rice cooker.
Recipes, categories, commetens and ratings are all stored in an API and API calls are managed in src/services/api.js
For details on API access, see the Environment section below.

This Project has been developed as a group project and is part of the Frontend with Frameworks module of the FUX25 program at Chas Academy, Stockholm.

## Environment

For access to API a teamId is required. If you do not have a teamId, please contact the Product Owner.
